    @InterfaceAudience.Public
@InterfaceStability.Stable
public class Append
extends Mutation
    Performs Append operations on a single row.

    This operation ensures atomicty to readers. Appends are done under a single row lock, so write operations to a row are synchronized, and readers are guaranteed to see this operation fully completed.

    To append to a set of columns of a row, instantiate an Append object with the row to append to. At least one column to append must be specified using the add(byte[], byte[], byte[]) method.
