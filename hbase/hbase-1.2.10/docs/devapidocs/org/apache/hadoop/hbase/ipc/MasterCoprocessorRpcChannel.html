    @InterfaceAudience.Private
public class MasterCoprocessorRpcChannel
extends CoprocessorRpcChannel
    Provides clients with an RPC connection to call coprocessor endpoint Services against the active master. An instance of this class may be obtained by calling HBaseAdmin.coprocessorService(), but should normally only be used in creating a new Service stub to call the endpoint methods.
