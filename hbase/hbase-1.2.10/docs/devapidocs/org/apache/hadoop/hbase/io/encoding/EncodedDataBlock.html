Class EncodedDataBlock

  • java.lang.Object
    • org.apache.hadoop.hbase.io.encoding.EncodedDataBlock

  • @InterfaceAudience.Private
public class EncodedDataBlock
extends Object
    Encapsulates a data block compressed using a particular encoding algorithm. Useful for testing and benchmarking. This is used only in testing.
