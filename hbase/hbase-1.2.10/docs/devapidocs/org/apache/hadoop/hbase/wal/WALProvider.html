org.apache.hadoop.hbase.wal

Interface WALProvider

  • All Known Implementing Classes:
    BoundedRegionGroupingProvider, DefaultWALProvider

    @InterfaceAudience.Private
public interface WALProvider
    The Write Ahead Log (WAL) stores all durable edits to the HRegion. This interface provides the entry point for all WAL implementors.

    See DefaultWALProvider for an example implementation. A single WALProvider will be used for retrieving multiple WALs in a particular region server and must be threadsafe.
