Class RateLimiter

  • java.lang.Object
    • org.apache.hadoop.hbase.quotas.RateLimiter
  • Direct Known Subclasses:
    AverageIntervalRateLimiter, FixedIntervalRateLimiter

    @InterfaceAudience.Private
@InterfaceStability.Evolving
public abstract class RateLimiter
extends Object
    Simple rate limiter. Usage Example: // At this point you have a unlimited resource limiter RateLimiter limiter = new AverageIntervalRateLimiter(); or new FixedIntervalRateLimiter(); limiter.set(10, TimeUnit.SECONDS); // set 10 resources/sec while (true) { // call canExecute before performing resource consuming operation bool canExecute = limiter.canExecute(); // If there are no available resources, wait until one is available if (!canExecute) Thread.sleep(limiter.waitInterval()); // ...execute the work and consume the resource... limiter.consume(); }
