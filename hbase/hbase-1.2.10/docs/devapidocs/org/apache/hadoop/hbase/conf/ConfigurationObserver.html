Interface ConfigurationObserver

  • All Known Subinterfaces:
    LoadBalancer, PropagatingConfigurationObserver, Region, Store
    All Known Implementing Classes:
    BaseLoadBalancer, CompactSplitThread, FavoredNodeLoadBalancer, HRegion, HStore, MasterRpcServices, RpcServer, RSRpcServices, SimpleLoadBalancer, StochasticLoadBalancer

    @InterfaceAudience.Private
@InterfaceStability.Evolving
public interface ConfigurationObserver
    Every class that wants to observe changes in Configuration properties, must implement interface (and also, register itself with the ConfigurationManager object.
