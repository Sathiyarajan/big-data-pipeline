      • taskLoop

        public void taskLoop()
              throws InterruptedException
        Wait for tasks to become available at /hbase/splitlog zknode. Grab a task one at a time. This policy puts an upper-limit on the number of simultaneous log splitting that could be happening in a cluster.

        Synchronization using taskReadyLock ensures that it will try to grab every task that has been put up
