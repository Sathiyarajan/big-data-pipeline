org.apache.hadoop.hbase.zookeeper

Class ZKTableStateClientSideReader

  • java.lang.Object
    • org.apache.hadoop.hbase.zookeeper.ZKTableStateClientSideReader

  • @InterfaceAudience.Private
public class ZKTableStateClientSideReader
extends Object
    Non-instantiable class that provides helper functions to learn about HBase table state for code running on client side (hence, not having access to consensus context). Doesn't cache any table state, just goes directly to ZooKeeper. TODO: decouple this class from ZooKeeper.
