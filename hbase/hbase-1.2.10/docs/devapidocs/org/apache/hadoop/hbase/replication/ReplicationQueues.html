org.apache.hadoop.hbase.replication

Interface ReplicationQueues

  • All Known Implementing Classes:
    ReplicationQueuesZKImpl

    @InterfaceAudience.Private
public interface ReplicationQueues
    This provides an interface for maintaining a region server's replication queues. These queues keep track of the WALs that still need to be replicated to remote clusters.
