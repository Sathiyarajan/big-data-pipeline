org.apache.hadoop.hbase.replication

Interface ReplicationQueuesClient

  • All Known Implementing Classes:
    ReplicationQueuesClientZKImpl

    @InterfaceAudience.Private
public interface ReplicationQueuesClient
    This provides an interface for clients of replication to view replication queues. These queues keep track of the WALs that still need to be replicated to remote clusters.
