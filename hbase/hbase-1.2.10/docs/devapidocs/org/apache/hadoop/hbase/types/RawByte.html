Class RawByte

  • java.lang.Object
    • org.apache.hadoop.hbase.types.RawByte
  • All Implemented Interfaces:
    DataType<Byte>

    @InterfaceAudience.Public
@InterfaceStability.Evolving
public class RawByte
extends Object
implements DataType<Byte>
    An DataType for interacting with values encoded using Bytes.putByte(byte[], int, byte). Intended to make it easier to transition away from direct use of Bytes.
    See Also:
    Bytes.putByte(byte[], int, byte)
